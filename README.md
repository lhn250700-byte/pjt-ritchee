# ![로고(마크)](https://github.com/lhn250700-byte/pjt-ritchee/blob/main/pjt-ritchee/public/logo.png?raw=true) 구로구 치과 리뷰 및 예약 시스템 릿치 (Ritchee)

> ### *Who are we?*
- 제작 기간 : 2025-11-17 ~ 2023-12-22
- 팀구성 : 김훈규, 박종석, 임윤섭, 장희란, 이하늘, 홍지승

### [프로젝트 노션링크](노션 링크 들어갈 자리)

<br />

> ### *Overview*
*릿치는 구로구 주민들이 치과를 선택할 때 광고나 단순한 입소문이 아닌,*

*실제 진료 경험을 바탕으로 신뢰할 수 있는 치과를 선택 할 수 있도록 돕는 예약, 리뷰 서비스입니다.*

*예약부터 진료, 리뷰까지 하나의 흐름으로 연결된 구조를 통해 의료 정보의 신뢰도를 높이고자 했습니다.*
